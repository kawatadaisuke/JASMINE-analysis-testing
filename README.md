### JASMINE mapping strategy

#### JASMINE mapping simulation and evaluation of astrometric accuracies.

To run the simulation,

``` console
> cd mapping-strategy/obsplan/
> python MPSv?.py

```

To make plots, use `Plot_MPS_results.ipynb`
